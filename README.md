# BetterGR Web Application

A modern web application built with Next.js and TypeScript, featuring a beautiful UI powered by Tailwind CSS and Radix UI components.

## Technologies

- **Framework**: [Next.js 15.2.0](https://nextjs.org/) - React framework for production

### Prerequisites

- Node.js (LTS version recommended)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/betterGR-web-app.git
cd betterGR-web-app
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Run the development server:
```bash
npm run dev
# or
yarn dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run start` - Start production server
- `npm run lint` - Run ESLint
